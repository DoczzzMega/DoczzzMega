### Quick Intro

I'm a self-taught full stack developer with over 3 years of experience — all from free resources. I ship web applications from concept to production: frontend in **React/Vue**, backend in **Laravel**, data in **MySQL/Redis**, containers with **Docker**, deployed on **Ubuntu**.

I understand how web application vulnerabilities are exploited in practice, and I use this knowledge to write secure code from the start.

**Now:** AI-powered web applications.
